# DATA_SCHEMA.md - Variable & Data Structures (v2.5)

## 1. 가이드 페이지 디자인 통일 규칙
사용자가 데이터의 비중을 명확히 인지할 수 있도록 가이드의 모든 분석 지표는 동일한 스타일을 적용한다.
- **Highlight Box**: 모든 지표의 최상단에는 "실제 통계 결과"를 명시한 `.stat-highlight` 박스를 배치한다.
- **List Highlight**: 홀짝 및 고저 리스트는 `.top-logic-list` 클래스를 통해 배경색과 테두리를 부여하여 일반 텍스트와 구분한다.
- **Fraction Display**: 모든 확률 뒤에 반드시 `(횟수/전체)` 분수 표기를 유지한다.

## 2. 주요 통계 합산 항목 (Guide)
- **Core Ratios**: 홀짝/고저 분석 시 가장 높은 당첨 빈도를 보이는 `4:2, 3:3, 2:4` 비율의 합계를 별도로 계산하여 최상단에 강조 노출한다.
