# DATA_SCHEMA.md - Variable & Data Structures (v2.4)

## 1. 실시간 툴팁(Tooltip) 데이터 규칙
메인 화면의 각 분석 지표에 대한 공약 팁을 `index.html`의 `data-tip` 속성에 정의한다.
- **표시 조건**: 마우스 오버(Hover) 시 해당 지표의 의미와 최적 전략을 툴팁으로 노출.
- **스타일 정책**: CSS `::after` 가상 요소를 사용하며, `z-index: 5000`을 부여하여 최상단 노출 보장.

## 2. 툴팁 주요 문구 예시
- **총합**: "역대 당첨 70% 이상이 총합 120~180 사이에서 출현합니다."
- **이월수**: "직전 회차 번호가 1~2개 다시 나올 확률이 매우 높습니다."
- **AC값**: "산술적 복잡도(AC)는 7 이상일 때 실제 무작위 번호에 가깝습니다."
