# 로또 번호 분석 및 추천 서비스 PRD (v2.9)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **고도화된 동적 가이드 (Enhanced Dynamic Guide)**: 
    - 각 분석 지표별 **세이프 존(Safe Zone) 실제 적중률** 정보를 확률(%)과 분수(횟수/전체) 형식으로 실시간 제공.
    - 공략 팁 영역에 해당 지표의 **권장 세이프 수치 범위**를 동적으로 주입하여 사용자 의사결정의 편의성 극대화.
- **가중치 기반 파레토 분석 (G0)**: 최근 20회차 흐름에 60% 가중치를 부여하여 현재 시점에서 가장 확률이 높은 '진짜 골드/실버' 영역 추출.
- **전략적 번호 생성 (2:3:1)**: 최근 적중률이 높은 실버존 비중을 강화한 골드(2):실버(3):일반(1) 하이브리드 조합 알고리즘 제공.
- **정규분포 기반 지표 분석 (G1~G5)**: 
    - **옵티멀 존 (Optimal)**: 평균 ±1σ 범위 (역대 데이터의 68% 밀집).
    - **세이프 존 (Safe)**: 평균 ±2σ 범위 (역대 데이터의 95% 유효).
- **지능형 시각화**: 전 화면(메인, 히스토리, 통계, 가이드) 일관된 색상 코드 적용 및 차트 레이블 최적화.

## 3. 기술적 완성도
- **실시간 데이터 바인딩**: 가이드 페이지를 포함한 전 서비스 영역이 `advanced_stats.json`의 통계 원천 데이터를 동적으로 소비함.
- **무결성 데이터 파이프라인**: Python 분석 엔진과 JS 렌더링 엔진 간의 완벽한 데이터 동기화.
