# 로또 번호 분석 및 추천 서비스 PRD (v3.3)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **원스톱 분석 프로세스 (One-stop Analysis)**: 조합 분석기에서 자동/반자동 선택 시 번호 생성과 동시에 분석 리포트가 즉시 활성화되도록 UX 흐름 단축.
- **지능형 반자동 선택 (Intelligent Semi-auto)**: 사용자가 고정하고 싶은 번호를 선택한 후 [반자동 선택] 클릭 시, 나머지 슬롯을 가중치 기반 파레토 영역(Gold/Silver)에서 우선 추출하여 채워주는 기능 제공.
- **직관적 공략 팁 (Actionable Tips)**: 가이드 페이지의 공략 팁을 `[지표명] 권장 세이프 "[범위]" 이 좋습니다.` 형식으로 표준화.
- **전문 통계 데이터 렌더링**: 가이드 페이지 내 통계 정보를 압축 형식(`범위 확률%(횟수/전체)`)으로 시각화.

## 3. 기술적 완성도
- **통합 데이터 파이프라인**: 모든 수치 지표에 대해 정규분포 기반의 SD 산출 및 영역 히트수 실시간 계산 엔진 구현.
- **시각적 일관성**: 전 화면 통합 컬러 시스템 및 실시간 동적 툴팁 반영.
