# 로또 번호 분석 및 추천 서비스 PRD (v2.2)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계 및 파레토 법칙(80:20)으로 정밀 분석하고, 사용자에게 과학적인 데이터 기반의 전략적 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **파레토 법칙 기반 영역 구분**: 출현 빈도 상위 20%를 '골드존', 상위 50%를 '실버존'으로 정의하여 데이터 집중도 시각화.
- **전략적 번호 생성 (2:2:1:1)**: 단순히 무작위가 아닌, 골드존(2개), 실버존(2개), 일반존(1개), 콜드존(1개)의 황금 비율 조합 제공.
- **5대 표준 분류 (G1~G5)**: 전 서비스의 분석 지표를 5개 그룹으로 표준화하여 일관된 분석 흐름 제공.
- **지능형 고정 헤더**: 실시간 번호와 영역별 분석 범례를 상단에 고정하여 페이지 어디서든 핵심 정보를 놓치지 않게 함.
- **정밀 내비게이션 (Vertical Centering)**: 메인 화면에서 통계 클릭 시, 해당 지표의 제목과 차트가 화면의 상하 정중앙(50% 지점)에 오도록 설계하여 정보 주목도를 극대화함.

## 3. 기술적 완성도
- **실시간 통계 가이드**: 정적 텍스트가 아닌 실제 역대 데이터 분포를 실시간으로 계산하여 가이드 내용에 반영함.
- **무결성 데이터 파이프라인**: Python 스크립트로 생성된 단일 JSON 원천을 전 화면이 완벽하게 공유 및 동기화함.
