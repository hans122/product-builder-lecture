# 로또 번호 분석 및 추천 서비스 PRD (v3.4)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **통합 상태 배지 시스템 (Unified Status Badges)**: 조합 분석 리포트의 모든 지표에 대해 실시간 통계($\mu \pm \sigma$) 기반의 최적(Optimal)/세이프(Safe)/주의(Warning) 상태 배지를 적용하여 시각적 분석 일관성 확보.
- **원스톱 분석 프로세스 (One-stop Analysis)**: 조합 분석기에서 번호 생성과 동시에 분석 리포트가 즉시 활성화되도록 UX 흐름 최적화.
- **지능형 반자동 선택 (Intelligent Semi-auto)**: 사용자 선택 번호를 유지하며 나머지 슬롯을 가중치 기반 파레토 영역에서 자동 추출.
- **전문 통계 데이터 렌더링**: 가이드 페이지 내 통계 정보를 고밀도 압축 형식으로 시각화.

## 3. 기술적 완성도
- **정밀 데이터 매핑**: `stats_summary` 내의 개별 수치 지표(odd_count, low_count 등)를 분석 로직과 1:1로 정확히 매핑하여 분석 정확도 극대화.
- **시각적 일관성**: 전 화면 통합 컬러 시스템 및 실시간 동적 툴팁 반영.
