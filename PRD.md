# 로또 번호 분석 및 추천 서비스 PRD (v3.18)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **자기 유지보수 데이터 파이프라인 (Self-Maintenance)**: 매주 로또 추첨 데이터 자동 수집 및 통계 엔진(`analyze_data.py`) 연동을 통한 무인 운영 시스템 구축.
- **컴포넌트 기반 UI 엔진 (LottoUI)**: 모든 시각적 요소(공, 지표, 카드)를 코어 엔진에서 생성하여 전사적 디자인 일관성 및 수정 효율성 극대화.
- **성능 최적화 및 Skeleton UI**: 데이터 로딩 지연 시 스켈레톤 애니메이션을 제공하여 사용자 체감 속도 향상 및 시각적 안정성 부여.
- **데이터 기반 자동화 아키텍처 (Data-Driven Automation)**: 전 페이지의 지표 렌더링 로직을 `INDICATOR_CONFIG` 설정 기반으로 통합.
- **이중 빗금 시각화 (Dual-Hatching)**: 세이프 존(-45° 파랑)과 옵티멀 존(45° 녹색)의 격자 교차를 통해 통계적 밀집도 표현.
- **정밀 통계 라벨링 (Precision Labeling)**: 모든 차트 하단에 6대 핵심 통계 지점 색상별 자동 마킹.
- **AI 스마트 조합 및 새로고침**: 통계적 필터링을 거친 Top 5 조합 제안 및 실시간 메인 지표 연동.
- **나의 조합 정밀 분석 (My Lab)**: `combination.html`에서 사용자가 직접 선택한 번호의 통계적 우수성을 점수와 등급으로 평가. 메인 허브의 추천 조합과 클릭 한 번으로 연동되어 정밀 지표 확인 가능.
- **실시간 데이터 동기화 (Shared Persistence)**: 메인 페이지의 추천 클릭이나 조합 분석 실행 시, 해당 번호가 즉시 전 서비스의 '최근 번호'로 등록되어 통계 대시보드와 유기적으로 연동됨.
- **커뮤니티 및 댓글 기능 (Disqus Integration)**: 전 페이지 하단에 Disqus 댓글 시스템을 통합하여 사용자 간의 분석 정보 공유 및 소통 창구 마련.
- **조합 번호 데이터 보존 (Combination Persistence)**: 사용자가 조합 분석 페이지에서 선택하거나 생성한 번호를 브라우저에 자동 저장하여, 페이지 이동이나 새로고침 후에도 상태를 완벽히 유지.
- **표준 내비게이션 통합**: [🔮 AI 분석&예측]과 [🎯 나의 조합분석]을 핵심 2대 축으로 메뉴 구조 개편.
- **통합 분석 표준 시퀀스**: [G1]~[G5] 표준 그룹 순서를 동일하게 적용하여 분석 일관성 확보.

## 3. 기술적 완성도
- **아키텍처 고도화**: 지표 관리의 단일 소스 원칙(SSOT)을 준수하는 루프 기반 렌더링 엔진 구축.
- **소셜 인터랙션**: 페이지별 고유 ID(`page.identifier`)를 활용한 독립적인 댓글 스레드 관리.
- **상태 복구 메커니즘**: `localStorage`를 연동하여 수동/자동 번호 상태 및 투표용지 마킹 UI를 동적으로 복구.
- **시각적 일관성**: 전 화면 통합 컬러 시스템 및 실시간 동적 툴팁 반영.
