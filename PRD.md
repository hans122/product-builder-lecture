# PRD (Product Requirement Document) - v22.0

## 1. 프로젝트 비전
로또 6/45 및 연금복권 720+ 데이터를 기반으로 과학적이고 정밀한 통계 지표를 제공하며, 전문가급 데이터 가시화와 10대 AI 전략 모델을 통해 사용자의 전략적 복권 구매를 돕는 통합 분석 플랫폼.

## 2. 핵심 타겟 유저
- 데이터 기반의 논리적 조합을 선호하는 로또/연금 분석가.
- 각 자리수별 출현 확률과 패턴을 중시하는 연금복권 유저.
- 통계적 유의미성과 시각적 가시성을 중시하는 데이터 전문가.

## 3. 주요 기능 요구사항
### 3.1. 로또 6/45 분석 (GL1~GL6)
- **GL1 기본 밸런스**: 총합, 홀짝, 고저 비율 분석.
- **GL2 회차 상관관계**: 이월수, 이웃수, 1~3회전 윈도우 매칭 및 연속번호 쌍 분석.
- **GL3 수적 속성**: 소수, 합성수, 3/5배수, 제곱수 및 쌍수 검증.
- **GL4 공간적 배치**: 3/5/9/15분할 구역 균형, 색상수, 모서리/삼각형 패턴 분석.
- **GL5 정밀 통계**: 끝수합, 동끝수, AC값 및 Span(간격) 분석.
- **GL6 고급 심층 지표**: 평균 간격(Mean Gap) 및 회귀 에너지 모형 적용.

### 3.2. AI 10대 예측 전략 (v22.0 고도화)
- **💎 다차원 최적화**: 모든 지표가 평균값에 수렴하는 정석 조합.
- **📊 패턴 유사도형**: 최근 10회차 당첨 흐름 상관관계 반영.
- **🔥 기세 추종형**: 최근 출현 빈도가 높은 'Hot' 번호 집중.
- **⚖️ 밸런스 가중형**: 대칭적 균형미(홀짝/고저) 최적화.
- **🛡️ 데이터 방어형**: 장기 미출현 번호(Cold) 전략적 포함.
- **🚀 회귀 에너지형**: AI 출현 임박 지표 점수 기반 구성.
- **🏠 이웃수 시너지**: 직전 회차 연계성 극대화 조합.
- **🔢 소수/합성수형**: 수학적 확률 분포 밸런스 설계.
- **🎨 섹션 컬러 배분**: 전 구간 균등 분포 안정 조합.
- **📉 저빈도 역습형**: 희귀 패턴 기반의 공격적 잭팟 전략.

### 3.3. v22.0 신규 지능형 모듈
- **⚡ AI 회귀 타이밍 시스템 (Regression Timing)**: 각 분석 지표가 평균으로 회귀하려는 성질을 수치화(Energy %)하여 출현 임박 신호 제공.
- **📊 전수 빈도 시각화 (Frequency Chart)**: 1~45번 전체 번호의 출현 빈도를 히스토그램 형태로 제공하여 한눈에 파악 가능.
- **🧠 다층 데이터 통합 (Multi-layer Integration)**: 마르코프 체인과 회귀 에너지를 결합한 고차원 예측 알고리즘 적용.

### 3.4. 연금복권 720+ 분석 (GP1~GP13)
- **GP4 합계 및 균형 (Golden Zone)**: 6자리 합계 20~35 구간 분석.
- **GP9 역방향 체인 전략**: 1의 자리부터 당첨 확률을 누적하는 분석 기법.
- **GP13 당첨 흐름 타임라인**: 자리수별 이월 및 이웃 관계 시각적 추적.
- **전용 유틸리티 분리**: `pension_utils.js`를 통한 연금 특화 분석 알고리즘 독립.

### 3.5. 전문가급 시각화 표준
- **더블 라인 라벨링**: 차트 X축에 '실제 수치'와 '누적 확률(%)'을 상하 두 줄로 표기.
- **콘텐츠 밀착형 레이아웃**: 통계 표의 컬럼 너비를 데이터 크기에 최적화 (회차 55px, 당첨번호 230px, 결과값 80px).
- **중앙 정렬 정합성**: 모든 표 내부 데이터를 중앙 정렬하여 시각적 안정성 확보.
- **스켈레톤 로딩 (Pulse)**: 데이터 로딩 지연 시 사용자 경험을 위한 애니메이션 카드 제공.

## 4. 기술 스택 및 정책
- **Architecture**: Modular Hub (core.js / pension_utils.js / ui_components.js).
- **Standard**: Toss/KakaoBank 스타일의 고대비 핀테크 디자인 시스템 준수.
- **Sync Policy**: 시스템 버전(`v22.0`) 기반 리소스 캐싱 및 문서 동기화 강제.
