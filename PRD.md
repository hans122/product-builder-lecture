# 로또 번호 분석 및 추천 서비스 PRD (v1.1)

## 1. 프로젝트 개요
역대 로또 당첨 데이터를 기반으로 통계적 패턴을 분석하고, 이를 바탕으로 사용자에게 최적화된 로또 번호 조합을 추천 및 분석해 주는 웹 서비스입니다.

## 2. 목표 및 가치
- **데이터 기반 의사결정**: 단순 운이 아닌 통계적 확률에 근거한 번호 선택 가이드 제공.
- **사용자 경험 극대화**: 직관적인 차트와 실시간 분석 결과를 통해 번호의 '품질'을 즉시 확인.
- **심화 분석 제공**: 일반적인 홀짝/고저를 넘어 구간 분할, AC값, 누적 이월 확률 등 전문적인 지표 제공.

## 3. 주요 기능 (Features)
### 3.1. 번호 생성 및 실시간 분석 (메인)
- 버튼 클릭 시 1~45 중 중복 없는 6개 번호 생성 및 자동 정렬.
- 생성된 번호에 대한 **실시간 통계 지표** 즉시 표시:
    - 기본: 홀짝, 고저, 총합.
    - 상관관계: 직전 회차 출현(이월수), 주변번호(이웃수), 연속번호 쌍.
    - **특수**: 이월수 누적 출현 확률(1~2개, 1~3개 범위).
    - **심화**: 3/5/9/15 구간 분할 점유율, 5색상 분할 분석.
    - 전문: AC값(산술적 복잡도), Span(번호 간격), 용지 패턴(모서리, 삼각형).

### 3.2. 역대 통계 대시보드
- 항목별 역대 출현 빈도 및 분포 차트 시각화.
- 각 차트 옆에 최근 회차의 실제 데이터를 미니 테이블로 제공하여 흐름 파악 지원.
- 번호별 전체 출현 빈도 TOP 10 제공.

### 3.3. 분석 히스토리
- 최근 30회차 당첨 번호의 모든 통계 지표를 테이블 형태로 제공.
- **이월수 심화 추적**: 이월수 총 개수와 함께 '1~2개', '1~3개' 범위 포함 여부를 숫자로 표시.
- 상단 요약 바를 통해 최근 회차의 이월수 분포 요약 정보 제공.

### 3.4. 이용 가이드 및 정밀 분석
- 각 통계 지표의 정의와 당첨 확률이 높은 '최적 범위' 설명.
- 실시간 데이터를 가이드 문구에 자동 반영하여 신뢰도 계산.

## 4. 분석 지표 기준 (Analytical Standards)
- **총합**: 120 ~ 180 (이상적 범위)
- **홀짝/고저**: 2:4, 3:3, 4:2 (균형 범위)
- **이월수**: 1~3개 포함 (통계적 다수 출현)
- **AC값**: 7 이상 (무작위성 확보)
- **구간 점유**: 다중 분할(3/5/9/15)을 통한 번호 분산도 측정.

## 5. 기술 스택
- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript.
- **Data Pipeline**: Python (CSV 전처리 및 JSON 생성).
- **Storage**: LocalStorage (최근 생성 번호 유지).
