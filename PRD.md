# 로또 번호 분석 및 추천 서비스 PRD (v3.12)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **조합 번호 데이터 보존 (Combination Persistence)**: 사용자가 조합 분석 페이지에서 선택하거나 생성한 번호를 브라우저에 자동 저장하여, 페이지 이동이나 새로고침 후에도 상태를 완벽히 유지.
- **표준 내비게이션 및 타이틀 통일 (Standardized Navigation & Titles)**: 전 화면의 메뉴 명칭과 타이틀을 동기화하고 헤더 레이아웃을 중앙 정렬로 최적화.
- **통합 분석 표준 시퀀스**: 전 화면에서 [G1]~[G5] 표준 그룹 순서를 동일하게 적용하여 분석 일관성 확보.
- **실전 공략 팁 제공**: 가이드 페이지에서 각 지표별 통계 기반 맞춤형 전략 노출.

## 3. 기술적 완성도
- **상태 복구 메커니즘**: `localStorage`를 연동하여 수동/자동 번호 상태 및 투표용지 마킹 UI를 동적으로 복구.
- **정밀 데이터 매핑**: `stats_summary` 내의 모든 원시 지표 키를 분석 로직과 1:1 매핑.
- **시각적 일관성**: 전 화면 통합 컬러 시스템 및 실시간 동적 툴팁 반영.
