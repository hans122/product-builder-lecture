# 로또 번호 분석 및 추천 서비스 PRD (v3.0)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **전문 통계 데이터 렌더링**: 가이드 페이지 내 통계 정보를 `옵티멀 존 범위 확률%(횟수/전체), 세이프 존 범위 확률%(횟수/전체)` 형식으로 압축 및 시각화하여 정보 밀도와 전문성 강화.
- **고정밀 수치 데이터 (Total Sum)**: 총합 분포 데이터를 기존 범주형에서 개별 수치형(Raw Numeric)으로 고도화하여 Span, 끝수합 지표와 동일한 정밀 분석 차트 제공.
- **동적 가이드 가이드라인**: 공략 팁 영역에 실시간 계산된 **권장 세이프 범위**를 주입하여 사용자 조합의 가이드라인 제시.
- **가중치 기반 파레토 분석 (G0)**: 최근 20회차 흐름에 60% 가중치를 부여하여 현재 시점에서 가장 확률이 높은 '진짜 골드/실버' 영역 추출.
- **전략적 번호 생성 (2:3:1)**: 최근 적중률이 높은 실버존 비중을 강화한 조합 알고리즘 제공.

## 3. 기술적 완성도
- **통합 데이터 파이프라인**: 모든 수치 지표에 대해 정규분포 기반의 SD 산출 및 영역 히트수 실시간 계산 엔진 구현.
- **시각적 일관성**: 메인, 히스토리, 통계, 가이드 전 화면 통합 컬러 시스템(Green/Blue/Red) 적용.
