# 로또 번호 분석 및 추천 서비스 PRD (v2.5)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계 및 파레토 법칙(80:20)으로 정밀 분석하고, 사용자에게 과학적인 데이터 기반의 전략적 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **파레토 법칙 기반 분석 (G0)**: 최근 20회차 가중치를 반영한 골드/실버 영역 분석 가이드 제공.
- **정규분포 기반 지표 분석 (G1~G5)**: 
    - **옵티멀 존 (Optimal)**: 평균 ±1σ 범위 (역대 데이터의 68% 밀집).
    - **세이프 존 (Safe)**: 평균 ±2σ 범위 (역대 데이터의 95% 유효).
- **전략적 번호 생성 (2:3:1)**: 최신 트렌드를 반영하여 골드존(2개), 실버존(3개), 일반존(1개)의 최적 조합 제공.
- **22대 표준 지표 히스토리**: 전 지표의 흐름을 한눈에 파악하는 고도화된 히스토리 테이블.

## 3. 기술적 완성도
- **실시간 통계 가이드**: 정적 텍스트가 아닌 실제 역대 데이터 분포를 실시간으로 계산하여 가이드 내용에 반영함.
- **무결성 데이터 파이프라인**: Python 스크립트로 생성된 단일 JSON 원천을 전 화면이 완벽하게 공유 및 동기화함.
