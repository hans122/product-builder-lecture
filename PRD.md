# 로또 번호 분석 및 추천 서비스 PRD (v3.7)

## 1. 핵심 가치 및 비전
역대 당첨 빅데이터를 5대 표준 분류 체계, 가중치 기반 파레토 법칙, 그리고 정규분포 이론으로 정밀 분석하여 사용자에게 가장 과학적인 조합 근거를 제시함.

## 2. 주요 기능 및 UX 정책
- **통합 상태 배지 시스템 (Unified Status Badges)**: 조합 분석 리포트의 모든 지표에 대해 실시간 통계($\mu \pm \sigma$) 기반의 최적(Optimal)/세이프(Safe)/주의(Warning) 상태 배지를 적용하여 시각적 분석 일관성 확보.
- **전 지표 통계 투명성 (Full Statistical Transparency)**: 가이드 페이지에서 주요 지표뿐만 아니라 1~3회전, 3의 배수, 다단계 구간 점유 등 **모든 하위 지표에 대한 옵티멀/세이프 존 실제 적중률**을 공개.
- **통합 자동/반자동 (Unified Auto/Semi-auto)**: 
    - 중복된 "전체 자동" 버튼을 삭제하고, "자동/반자동" 버튼 하나로 생성 로직 통합.
- **수동/자동 시각적 명확화 (Manual vs Auto Distinction)**: 
    - **수동 선택**: 투표용지 내 **진한 네이비 마킹**과 번호 리스트 내 **"MY" 뱃지**를 통해 사용자 선택임을 강조.
- **프리미엄 버튼 UI**: 핵심 기능 버튼에 그라디언트(Blue/Gold) 및 호버 인터랙션을 적용하여 직관적이고 고급스러운 UX 제공.

## 3. 기술적 완성도
- **정밀 데이터 매핑**: `stats_summary` 내의 개별 수치 지표(odd_count, low_count 등)를 분석 로직과 1:1로 정확히 매핑하여 분석 정확도 극대화.
- **시각적 일관성**: 전 화면 통합 컬러 시스템 및 실시간 동적 툴팁 반영.
