# 로또 번호 분석 및 추천 서비스 SDD (v1.6)

## 1. 프론트엔드 실시간 연동 설계
- **가이드 시스템 (`guide.js`)**: `distributions` 데이터를 분석하여 "가장 확률 높은 최적의 조건"을 텍스트로 자동 요약 생성.
- **분류 엔진**: HTML 클래스 및 섹션 구조를 G1~G5 표준 분류 체계로 일원화하여 코드 유지보수성 향상.

## 2. 알고리즘 정밀도
- **확률 계산**: `(항목 카운트 / total_draws) * 100` 공식을 전 지표에 적용.
- **다중 조건 필터링**: `Array.filter().reduce()`를 사용하여 복수 구간(예: 제곱수 1개 또는 2개인 확률) 합산 로직 구현.

# 로또 번호 분석 및 추천 서비스 PRD (v1.6)

## 1. 지능형 가이드 시스템
단순한 분석 방법론 제시를 넘어, 현재까지 쌓인 역대 당첨 데이터를 실시간으로 투영하여 "지금 이 순간의 최적 확률"을 사용자에게 안내함.

## 2. 전 서비스 지표 표준화
메인 생성기, 통계 대시보드, 히스토리 목록, 정밀분석 리포트가 모두 동일한 5대 표준 분류(G1~G5)에 따라 구성되어 사용자에게 일관된 분석 경험을 제공함.
