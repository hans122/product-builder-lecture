# 로또 번호 분석 및 추천 서비스 SDD (v2.1)

## 1. 헤더 및 내비게이션 고도화
- **Enhanced Sticky Header**: 통계 화면의 핵심 안내 문구("5대 표준 분류 체계...")를 `sticky-header` 내부로 통합하여 '내 최근 번호'와 함께 상단에 상시 고정함.
- **Vertical Center Alignment**: 앵커 클릭 시 섹션 제목이 뷰포트의 상하 정중앙(50% 지점)에 오도록 `scroll-margin-top: 50vh`를 적용하여 가독성을 극대화함.

## 2. UI 레이아웃 최적화
- 헤더 내부의 여백(`gap: 5px`) 및 안내 문구 폰트 크기(`0.8rem`)를 조정하여 고정 영역이 실제 콘텐츠를 과도하게 가리지 않도록 슬림하게 유지함.
