# 로또 번호 분석 및 추천 서비스 SDD (v1.4)

## 1. 프론트엔드 컴포넌트 구조
- **Dashboard (`main.js`)**: 실시간 윈도우 매칭 및 등급 계산 로직 최적화.
- **Deep Analysis (`analysis.js`)**: `renderMiniTables` 함수를 통해 전 지표(3/5/9/15분할 포함)의 최근 데이터를 시각화.
- **Detailed Report (`combination.js`)**: `statsData.last_3_draws` 구조에 기반한 점수 산출 로직 복구.

## 2. 데이터 처리 알고리즘
- **Window Matching**: 합집합 연산(`union`)을 통해 1~2, 1~3회전 중복 개수를 정확히 도출.
- **Error Handling**: DOM 요소가 없는 경우를 대비한 가드 클로즈(`if (!container) return`) 및 `null` 데이터에 대한 기본값(`-`) 처리 로직 강화.

# 로또 번호 분석 및 추천 서비스 PRD (v1.4)

## 1. 주요 기능 개선 사항
- **통계 정밀도 향상**: 3/5/9/15분할 모든 단계에 대해 차트와 상세 표(미니 테이블)를 제공하여 입체적 분석 지원.
- **실시간 피드백 강화**: 소수, 합성수, 배수 등 특수수 포함 여부를 생성 즉시 확인 가능.
- **안정성 확보**: 데이터 구조 변경 시에도 정밀 분석 및 가이드 기능이 중단되지 않도록 로직 안정화.
